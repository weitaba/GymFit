# Bench Press Reference

## Directory Structure

```
bench_press/
├── 1-setup/          # 躺姿: feet planted, head centered, arch
├── 2-unrack/         # 出杆: arm angle <90°, no shoulder protraction
├── 3-grip_width/     # 握距: forearms vertical at bottom
├── 4-grip_type/      # 握法: neutral wrists, full grip
├── 5-breathing/      # 呼吸: exhale eccentric, inhale concentric
├── 6-touch_point/    # 向心落点: bar at nipple line
└── 7-elbow_angle/    # 肘部角度: 45-60°
```

## 7 Checkpoints

| # | Checkpoint | Standard |
|---|-----------|----------|
| 1 | setup | feet planted, head centered, slight arch, back tight |
| 2 | unrack | arm-to-ground angle < 90°, no shoulder protraction, core braced |
| 3 | grip_width | forearms vertical to ground at bottom |
| 4 | grip_type | neutral wrists, full grip (thumb around bar) |
| 5 | breathing | exhale on eccentric, inhale before concentric |
| 6 | touch_point | bar at nipple line ± 2cm |
| 7 | elbow_angle | 45-60° between arm and torso |

Add .jpg/.png images or .mp4 videos in each checkpoint directory.
Each file represents one sub-item of the checkpoint.
